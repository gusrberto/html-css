Tipos principais de Formatação

Negrito / Destaque

Nesta frase há um termo em negrito com a tag <b> (não semântica).

Agora nesta frase temos um termo em destaque utilizando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase há um termo em itálico com a tag <i> (não semântica).

Agora nesta frase temos um termo em ênfase usando a tag <em>. (semântica).

Texto Marcado

É possível criar um texto marcado utilizando a tag <mark>.

Agora outro texto marcado com o uso da tag.

Texto grande e pequeno

Há um texto grande e um texto pequeno neste parágrafo.

Texto excluído / deletado

Para marcar um texto afim de mostrar que ele deve ser lido porém não considerado utilizamos a tag <del>.

Texto inserido

Para darmos ênfase e marcarmos um texto como inserido no texto utilizamos a tag <ins>.

Essa tag substituiu a antiga tag <u> (não semântica).

Texto sobrescrito

Para inserir números acima das letras como em x2+3 utilizamos a tag <sup>.

Texto subscrito

Agora para inserir número abaixo das letras como em H2O utilizamos a tag <sub>.
